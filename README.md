# Student Management System（C言語）

## 概要

C言語の基礎理解を目的として制作した、コンソール型の生徒管理システムです。
生徒情報（ID・氏名・年齢）の登録、一覧表示、検索機能を実装しています。

---

## 使用技術
- C言語
- 構造体（struct）
- ポインタ
- 配列管理
- 標準入出力（stdio.h, string.h）

---

## 主な機能
- 生徒情報の追加（学籍番号・名前・年齢）
- 一覧表示
- ID検索
- メニュー選択による操作

---

## 実装の工夫点

### セキュリティ面
- **バッファオーバーフロー対策**：`scanf("%s")` ではなく `fgets()` を使用
- **入力バッファの適切な管理**：数値入力後の改行文字をクリア

### 設計面
- 構造体でデータを一元管理
- ポインタを用いて配列へ直接アクセス
- `#define` による登録上限管理（100件）
- 機能ごとの関数分割で可読性を意識

### エラーハンドリング
- 登録上限チェック
- 該当データなし時のメッセージ表示
